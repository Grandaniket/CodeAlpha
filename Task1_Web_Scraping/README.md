# Web Scraping of Quotes Data | CodeAlpha Data Analytics Internship

## Problem Statement -
The objective of this project is to extract quote information from a website using web scraping techniques. The goal is to collect structured data such as quotes, authors and tags from multiple web pages and store it in a usable format for further analysis.

## Tools Used -
1. Python
2. Jupyter Notebook
3. Pandas (for storing scraped data)
4. Requests (for sending HTTP requests)
5. BeautifulSoup (for HTML parsing)

## Project Workflow -
1. Imported required Python libraries for web scraping.
2. Sent HTTP requests to the target website to access webpage content.
3. Extracted HTML content from the web pages.
4. Parsed HTML structure using BeautifulSoup.
5. Identified required HTML tags and classes containing quote details.
6. Extracted information such as quotes, author's name and tags.
7. Scraped data from multiple pages of the website.
8. Stored the extracted data in a Pandas DataFrame.
9. Exported the final dataset into a CSV file.

## Files -
1. *Task1_Web_Scraping.ipynb* - Jupyter Notebook containing the complete web scraping code and explanations.
2. *Quotes.csv* - CSV file containing the extracted quote data.
